九月记事

    十一长假，宛如一梦，但却足以回味，因而十月即将终结时，我方才能在这夜深时回望九月，看清在岁月中渐渐清晰的自己。

    幸赖三省依旧，我得以审视九月得失，以下便从生活、工作、学习三个方来认识自己：

  • 生活
    1. 收获：
      1. 情绪控制得以较好的修炼，本月基本上未产生消极情绪，与老婆生气很少，且大多数情况下能在较短时间内耐心交流，及时道歉， 重新回到融洽的氛围，创造出了一种彼此舒适的家庭氛围。
      2. 健身坚持的很好，继续保持，以利于自己的身体健康。
      3. 阅读蒋介石的日记，了解关于其背后的故事，再一次体会到自省的重要，也同样看到其在日记上的坚持，相信自己捡到三省这个 法宝后，并坚持有三月之多，在后续的人生中，也能不断坚持，不断自省并提高自己。
      4. 在同老婆看台湾的综艺节目时，渐渐修去自己先入为主视其低俗化的观点，实际上某些节目非常有用，也提醒自己做一个独立思考 的人，具有理想判断，不跟风。
      5. 使用Kindle阅读的感觉不错，本月阅读了许多有意思的书籍，除外，切至原版系统后，阅读英文书籍也方便许多，继续加油，无 处不阅读，无处不思考。
      6. 人的利益需要自己争取，从自我将一改老好人的做事态度，努力为自己争取最大的利益，为自己的人生负责。
      7. 阅读Hackers and Painters时，读到Money is not equal to wealth相关一节，我对金钱与财富的观念得以刷新，创造 财富。
    2. 不足：
      1. 认同老婆提出的建议：非交心之人，切记慎言，在言行上做到有深度，有理性，除外，尽量避免透露过多的经济状况。
      2. 阅读时存在紧迫心态，后续应当做适当调理，避免仓促，对于自己选择的读物，勿必做到精读，深读。
      3. 减少买彩票时所抱的一夜暴富心态，平淡视之为之，避免失落感。
      4. 近来抽烟较多，尽量控制抽烟量。
  • 工作
    1. 收获：
      1. 为自己做了决定，并向领导提出辞职的想法，预计今年年末前离开华为，寻找并实现自己的梦想，相信这是我新的生活的开端， 我期望接受更大的挑战。
      2. 辞职邮件写的不错，言简意赅，勉励自己。
    2. 不足：
      1. 偶尔存在抱怨的情绪，避免将离职的情绪透露给他人，做好自己即可。
      2. 和同事交流仍存在急躁的心态，这点后续勿必避免之，可以有性格，但不必强施于人。
  • 学习
    1. 收获：
      1. 阅读文档功力得以提高，对于SHELL/PYTHON(包括GOOGLE PYTHON STYLE GUIDE)/EMACS LISP/GNU TOOLS的理解上升了一个台阶， 在公司日常工作中，写了若干个脚本，提高工作次效率。
      2. 完成EMACS LISP第四章节的翻译，了结自己预定的承诺，继续坚持完成翻译工作。
      3. ZhaMiao网开发正常，完成了部分JS交互功能以及一些计划中的功能开发。
      4. 重整BLOG，切换为MUSE方式发布，移除与WORDPRESS的关系。
      5. 阅读大量关于GENTOO EBUILD的文档，目前能够阅读PORTAGE TREE中的EBUILD文档，并能进行修改，后续有需要时，维 护一个属于自己的OVERLAY。
      6. 完成EMACS IDO.EL中一个BUG的修复（十月注：后来发现EMACS 23中已经集成IDO-MODE，并已修复相关问题，修复方式 与我的实现基本一致）。
    2. 不足：
      1. 对于ZhaMiao网的开发，存在计划偏差，仍存在惰性，后续必须戒除克服，力争其尽快完成上线。

八月记事

    八月已央，时光再次悄然滑去，北京夏天的余热继续留在这初秋时节，于是在九月四日的这个周末，我开始例行记录八月的旧事，我之所以乐此不疲的旧事重提，显然从这种事情上得到了我认为还算值得的好处，即：认识我自己。

    当然，我显得记忆力不胜旧时，于是需要借助于记类似于日记之类的东西，我为其订了一个名字，称之为“每日三省”，这旧事的素材，因而也有了着落。

    依然承前方式，从生活、工作、学习三个方面，来分析自己的得与失，当然，这次稍微对顺序做了修订，将“生活”提至最前，在于我认为，生活才是生命的重中之重，工作乃是为了生活，是为其次。

  • 生活
    1. 收获：
      1. 生活中偶尔会有失落的情绪，甚至影响自己一天的心情，需要抱以正向乐观的态度，并强化自信心，尽快地将之消化遗忘。
      2. 班级论坛出现故障，直接电话服务提供商，主动要求争取自己的权利，很快搞定，因此这也说明必须要有维权意识，并加以行动， 才能争取到自己的利益，我是老好人的时代结束了！
      3. 和老婆继续保持良好的交流，感谢老婆在我生活中给予的支持与帮助。
      4. 继续良好地控制了自己的情绪，少数情况少仍会心急，继续保持。
      5. 阅读完了高行健先生的两部小说和一部印度小说，同时又发现了一个Metal的好论坛，收了几本不错的重金属专辑。
      6. 去医院看了一下鼻炎，得到了自己并非过敏性鼻炎的结果，后续身体不适，勿必及时到好医院检查，不可乱医。
      7. 在看国外电视时，经常见到四五十岁的人，还能爱能玩，值得自己借鉴和学习，希望二十年后，我也能像他们一样。
      8. 人不可能事事精通，因此只要做到认识自己的不足，并发现自己的长处，避开不足，而发挥上处，才能最大化创造价值。
    2. 不足：
      1. 要避免将自己尚未成熟的想法告诉他人，这同上月所提事前三思有相同的道理，后续必须努力做到。
      2. 要努力从悲观主导切换为乐观主导，譬如老婆的考试，我首先以为不告诉我的原因是由于考试结果不理想，实则是老婆全部 通过，而我完全多虑，忋人忧天了。
      3. 陪女友时间稍显不足，要继续保持周三同老婆放松一下的习惯，拒绝电脑。
      4. 外出就餐勿必按需点餐，避免滥花钱财。
      5. 阅读小说稍显心切，即然是一种娱乐或者休闲方式，不妨静下心来，慢慢咀嚼的好。
      6. 外出办事能力还有待提高，避免自己的胆怯心，需要继续放松一些。
  • 工作
    1. 收获：
      1. 周一后续不再加班，改为早起的方式，一周固定加两次班即可，通过自己正确的安排，为生活腾出更多的时间。
      2. 工作上的问题，要有追根问底的精神，对于技术细节，要以事实和标准说活，避免情绪化和不必要的争执。
    2. 不足：
      1. 工作上的事情，需要把握当责与推责的边界，属于自己的责任，则不必含糊，不属于自己的责任，要据理力争。
      2. 继续优化心态，做到100%坦然面对他人指摘，错则接受，无错则力争之。
      3. 注意玩笑的度，避免开可能伤害他人的玩笑。
      4. 沟通技巧有待提高，但相比这前提升明显，语言需清晰有条理，但又不失幽默。
  • 学习
    1. 收获：
      1. “天下事有难易乎，为之，则难者亦易矣，不为，则易者亦难矣”，本月顺利重写网站开发计划，并实现了几个重要的功能点， 另外，阅读技术文档更为专心，因而消化速度也更快，于是顿悟这句话，收获很大。
      2. 完成Elisp number.texi一节的翻译，本月因时间关系，未投入第四章节的翻译，九月继续投入执行。
      3. 较为认真地阅读了Linux(Gentoo)的ebuild编写指南，另外阅读了gnu findutils的文档，收获很大。
      4. Python的语法特性，已经掌握大约有70%，并编译了Elisp和Python两版的网站一键上传工具，显得Python编写代码要更 快一些。
      5. 保持了勤作记录的习惯，遇到好的内容，优先记录在个人网站上，以备后续查阅。
    2. 不足：
      1. 学习起来有些惶惶，过于要求自己，经常出现研究东西至深夜的情形，而彼时自己非常之困，后续要先身体，后学习。
      2. 存在在网络上闲逛的情况，需要将这种情况减少一些，提高自控能力。

执子之手，与子偕老

    谨以此日志，记念我与萍走过的坎坷两年，并献给我们一定可得的美好未来。

    写这篇日志的时候，我突然想起“不能承受的生命之轻”里的特雷莎，萍就像特蕾莎一样，在没有预兆的情况下，投奔了我，并走入了我的生活；不过，我并不像托马斯，中国也不像那时的捷克，我与萍在坎坎坷坷中，经历了风风雨雨，走过了这两年，步入生活的正确轨道。

    我面前的书堆上，放着萍花了两月一针一线制成的送我的十字锈，萍对我讲它的意义，我尤其记得她谈到从无到有的兴奋，而我何尝不是如此，看着萍从两年前懵懂的小女孩，到如今通过了各种各样的考试并有着自己独特思想的女人，我也一样的兴奋。

    我们的生活，其实非常的简单与普通，彷佛旧时那种“男耕女织”，每日早出晚归的我，在家学习并期盼我归来的萍，所谓爱情，早已经融化在日常琐碎的关心中，我更相信的，就是“执子之手，与子偕老”，“死生契阔，与子成说”。

七月记事

    由于班级所在域名备案被注销，我也确实被我天朝上国的政策折腾的不轻，于是迫不得已，在近八月中旬时，才能将七月记事发布至博客上，不过这总结，倒是我在七月末所写，所谓有备无患，胸有成竹，大抵如此。

    闲话不说，七月记事继续沿用先前方式，分几个维度逐一总结，我特别想感谢地是自己的老婆，及时指出我的毛病，使得七月里我得到很大的启发与收获。

  • 工作：
    1. 收获（本月工作上收获较少，八月重点关注一下）：
      1. 工作中情绪掌控良好，并能根据具体情况，合适的调整语气以及语速。
    2. 不足：
      1. 要避免说话时存在的两个极端，一为过于含糊，如“估计”、“差不多”，二为过于肯定，如“绝对”、“肯定”，除外，要避免“我以为”。
      2. 技术钻研时偶尔代还存在分心，需要避免这种跳跃的思维，专注于将事情阶段化。
      3. 工作中也同样需要用心，如同近日所读文章所说，要用脑子做事，而非简单地用手做事。
      4. 对于徒弟的关注不够，后续要继续纠正其工作作风，促其做事更为准确高质量。
  • 生活：
    1. 收获：
      1. 逐渐意识到老婆对自己的意义，我们共同生活了三年，老婆对于我性格的拿捏，以及对于我缺点的把握，都非常之到位，是我生活道路上真正的伴侣。
      2. 陪老婆检查完眼睛，了却了自己对于老婆的承诺，人这一生，并不能十全十美，但总要相信，上帝会给你打开另一个扇门的，只要你愿意靠近这门口，祝福老婆在未来的生活中更加成功。
      3. 生活中情绪控制很好，本月基本未同老婆争吵，即便有彼此不认同的地方，也能相互心平气和地指出并消解矛盾。
      4. 逐渐修正并移除了攀比心理，过自己的生活，着重于自身的积累与自由。
      5. 阅读了三部小说以及李连杰传，特别是《基督山恩仇录》一书，对我触动很大，也更让我明白了等待与坚持的意义，见个人网站“读书”页面。
      6. 生活中能够通过ORG-MODE有效管理自己的待办事项，本月生活上基本上没有遗漏的事情，均及时完成。
      7. 读到了一句非常有禅意的话“例如和尚烧一大壶水，除加柴外，还可考虑减水，这样才能尽快将水烧开”，我不必总想一口吃个胖子，生活应该是一个逐渐筛选目标并实现目标的过程，人生很长， 不必事事都投入，何况人的精力本身就很有限。
      8. 对于做事，一定要细而稳，事事都抱着浅尝辄止的心态，又岂会成功。
      9. 和老婆去天津做了一次旅行，非常放松心情，天津名物甚多，大饱眼福，今年下几站，南京、苏杭以及上海，期待成行。
    2. 不足：
      1. 吉他练习本月投入时间很少，四次周末仅有一次周末有时间，但又处理了卫星天线，八月有时间可再做投入。
      2. 总体上陪老婆的时间偏少，但后面装好卫星天线后，陪老婆看了几次电视，感觉不错，这个可以做为一个两个人休闲的方式。
      3. 要尽力学会忽视别人对自己的影响，注定与某些人非同路人，保持一般的浅浅关系即可，不必强求。
      4. 努力多作些家务，一则为锻炼，一则为老婆分担一些家务压力。
      5. 睡前如不能及时入睡，可考虑多与自己做做交流，思考一些关于未来生活的计划与安排，为后续自己独立创业做计划。
  • 学习：
    1. 收获：
      1. ZHAMIAO(PYLONS)网站实现了30%的功能，熟悉了PYLONS开发的模式，成功配置SQLALCHEMY、ROUTE、认证等内容，对于PYTHON语言，有了进一步的了解， 八月应当继续进行优化开发，争取年底能实现60%功能。
      2. ELISP翻译本月投入较少，但仍完成了第二章节的翻译，第三章节整数部分已经完毕，八月争取完成第三章节与第四章节的翻译。
    2. 不足：
      1. 在研究技术时，要能够静心并且入心，追求更为深入地理解。
      2. 总有一些懒惰情绪，不能够主动去完成未完之事，这点亟待解决改正。
      3. 避免心血来潮做事情，坚持两三件事，用一年的时间，将这两三件事完成，要比一时兴起，一年未完成一件事情好的多。
  • 补充：

    以下内容为7月24日，也即同学自国外回京一聚后次日晚，我因酒醉未去上班，和老婆做了一番交流，老婆指出我的几个重要缺点，我非常之认可，故记录于此，以供后续参考：

    1. 做事需要充分思考，即事前三思，比如本月之处理卫星天线，初期风风火火，结果绕了很大的圈子，以后做事前一定充分查验。
    2. 不做老好人，要坚持自己的想法，不要轻易被别人说服，这个社会需要我们为自己的利益着想。
    3. 不大手大脚，不显摆，以免惯坏自己身边的人，没有义务为他人买单。
    4. 喝酒需要控制量度，醉可以，但不可过度，避免酒后失言。
