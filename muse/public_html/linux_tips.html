系统基础

    1）GNU findutils

    由于工作的原因，经常需要使用到find和xargs这两个命令，后来抽空在网上查阅了相关资料，find以及xargs等命令，由一个GNU findutils的包提供， 该包的文档在GNU网站上已经提供，请访问http://www.gnu.org/software/findutils/manual/html_mono/find.html查阅。

    常用命令收集：

  • 查找当前目录下包含"pattern"的目录，并将其移动至"/mnt/dest"目录下，这里的 -I XX表示在xargs的初始命令中用xargs的输入替换XX。

      find . -name "pattern*" -type d | xargs -I XX mv XX /mnt/dest

    2）ANSI color（让你的SHELL脚本更加多彩）

    维基百科上关于ANSI color的介绍

    借助于上面介绍的内容，可供你写出诸如如下这样多彩的shell脚本：

    # This will output the text in blue color.
    echo -e "\033[34mWelcome to the blue color wolrd"

    # This will output the text in red color.
    echo -e "\033[31mWelcome to the red color world"

Gentoo

    1）Gentoo OpenRC迁移指导

    最近偶然间读到Gentoo OpenRC的迁移指导，OpenRC用来取代Gentoo原有的以bash-shell编写的RC系统，并从baselayout1中剥离出来，和baselayout2成为独立的两个包，改用C写成，从而更方便的实现系统调用，以便支持更多的平台，如Gentoo/FreeBSD、Gentoo Embedded。

    其链接为：http://www.gentoo.org/doc/en/openrc-migration.xml
